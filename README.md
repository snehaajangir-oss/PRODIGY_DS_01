#  PRODIGY_DS_01

## Data Visualization using the Titanic Dataset

## Overview

This project was completed as **Task 01** of my **Data Science Internship** at **Prodigy InfoTech**.

The objective of this task was to create visualizations that represent the distribution of both continuous and categorical variables using the Titanic dataset. Through this project, I explored the dataset and created meaningful visualizations to better understand passenger demographics and survival patterns.

## Objective

- Create a histogram to visualize the distribution of a continuous variable.
- Create bar charts to visualize the distribution of categorical variables.
- Gain practical experience in Exploratory Data Analysis (EDA) using Python.

## Dataset

- **Dataset:** Titanic Dataset
- **Source:** Kaggle
- **Records:** 891 passengers

## Tools & Libraries Used

- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## Visualizations

- Histogram – Age Distribution
- Bar Chart – Passenger Class Distribution
- Bar Chart – Survival Distribution

## Key Insights

- Most passengers were between 20 and 40 years of age.
- The majority of passengers traveled in Third Class.
- More passengers did not survive than survived.
